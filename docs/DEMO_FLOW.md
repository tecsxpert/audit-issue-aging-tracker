# Demo Flow

## 1. Start Stack

```sh
docker compose up --build -d
docker compose ps
```

## 2. Show Health

```sh
curl http://127.0.0.1:8000/health
```

## 3. Run Automated Validation

```sh
python -m validation.e2e_test_runner
```

## 4. Demonstrate Security

```sh
python -m validation.security_validation
```

## 5. Demonstrate AI Value

Call all three AI endpoints using the JWT generated by validation helpers or the backend token issuer.
